Projeto - Lixeira Cata Bomba

Sobre o Projeto

Desenvolvi, junto com minha equipe, o projeto Lixeira Cata-Bomba, uma solução inovadora e gamificada que transforma o descarte de resíduos em uma experiência divertida. Como funciona? Criamos uma lixeira interativa com quatro compartimentos que representam bocas de monstros. Um dos compartimentos se ilumina aleatoriamente, desafiando o usuário a “atacar” o monstro acertando o lixo no local certo. A cada acerto, a lixeira emite sons dinâmicos, como o pavio de uma bomba queimando, e, ao final, simula uma explosão para simbolizar a vitória sobre o monstro. Esse projeto combina tecnologia Arduino, design criativo e sustentabilidade, incentivando o descarte correto de forma lúdica e engajante.

Durante o desenvolvimento, fui capaz de explorar várias tecnologias, como JavaScript, HTML5, CSS3 e algumas bibliotecas para criar uma experiência envolvente para o usuário.
